## 概要

サイトのトップページ (`/`) に AI チャットページを実装する。FUNE に自然言語で質問し、記事への導線を得られるインタラクティブな体験を提供する。

### 4. レスポンシブ対応

**デスクトップ** (lg 以上):
- ChatGPT ライクなレイアウト
- メッセージエリア: 中央揃え、最大幅制限 (max-w-3xl 程度)
- 吹き出しなし、シンプルなテキスト表示
- ユーザーメッセージ: 右寄せ
- FUNE の回答: 左寄せ

**モバイル** (lg 未満):
- LINE ライクな吹き出し UI
- FUNE アイコン表示（アバター）
- ユーザーメッセージ: 右側吹き出し
- FUNE の回答: 左側吹き出し + アイコン

**共通**:
- 入力欄は画面下部に固定（`position: sticky; bottom: 0`）
- タグエリアは入力欄の上に常時表示
- 会話エリアはスクロール可能
- 新しいメッセージで自動スクロール

```typescript
  // type: text → テキスト追記
  // type: articles → 記事カード表示

  // type: suggestions → 関連タグ表示
  // type: done → 完了
  // type: error → エラーメッセージ
}
```

**ストリーミングテキスト表示**:
- テキストチャンクを受信するたびに画面に追記
- マークダウンをリアルタイムでレンダリング（react-markdown 等）
- タイピングアニメーション（カーソル点滅）

**ストリーミング完了後**:
- 記事カードをアニメーション付きで表示
- スポット情報カードを表示
- 関連タグ提案を表示

### 7. 会話状態管理

```typescript
interface ChatState {
  messages: ChatMessage[]        // 全メッセージ
  isStreaming: boolean           // ストリーミング中か
  sessionId: string              // 匿名セッション ID
}

interface ChatMessage {
  role: 'user' | 'assistant'
  content: string                // テキスト
  articles?: ArticleCard[]       // 関連記事
  spots?: SpotInfo[]             // スポット情報
  suggestions?: string[]         // 関連タグ
  timestamp: number
}
```

- `sessionStorage` にメッセージ履歴を保存
- タブを閉じたらリセット
- セッション ID は `localStorage` に保存（日次制限の識別用）

### 8. 入力エリア

- テキスト入力（`<textarea>` auto-resize）
- Enter で送信（Shift+Enter で改行）
- 送信ボタン（アイコン: Send / ArrowUp）
- ストリーミング中は送信ボタンを停止ボタンに変更
- 入力中は送信ボタンをアクティブ色に

### 9. 記事カード

既存の `PostCard` コンポーネントをベースに、チャット内での表示に最適化:

- コンパクトな横型レイアウト（サムネイル左、テキスト右）
- サムネイル + タイトル + 抜粋（2 行まで）
- クリックで記事ページへ遷移
- ホバーエフェクト

### 10. スポット情報カード

```
┌──────────────────────────┐
│ 📍 スポット名             │
│ 〒399-xxxx 長野県飯田市... │
│ TEL: 0265-xx-xxxx        │
│ 営業: 11:00〜21:00        │
│ [Googleマップで見る →]     │
└──────────────────────────┘
```

- 住所がある場合: Google マップリンクを生成（`https://maps.google.com/?q=住所`）
- 電話番号がある場合: `tel:` リンク
- 営業時間・備考がある場合: 表示

### 11. ローディング状態

- ストリーミング開始前: 「考え中...」+ ドットアニメーション
- ストリーミング中: カーソル点滅
- エラー時: エラーメッセージ + リトライボタン

### 12. アクセシビリティ

- `aria-live="polite"` でストリーミングテキストをスクリーンリーダーに通知
- 入力欄: `aria-label="FUNEに質問する"`
- キーボード操作: Tab でタグ間移動、Enter で送信

## デザイン方針

- 既存のデザインシステム（Tailwind + カスタム CSS 変数）を踏襲

- 記事カード: 既存の PostCard のスタイルを踏襲
